# ToolNest

ToolNest is a client-side web app that collects handy utilities in one place, including:

- Unit conversion
- Password generation
- QR code generation
- Base64 encoding/decoding
- Lorem ipsum text generation
- Color picker

## Tools included

- Unit converter
- Password generator
- QR code generator
- Base64 encoder/decoder
- Lorem ipsum generator
- Color picker

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## Notes

- All tools run entirely on the client side.
- AdSense placeholders are included in the UI.
